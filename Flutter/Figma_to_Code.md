### 2. Convert Figma Design into Flutter Code

### 3. 사용 후기
1. Figma UI를 그대로 가져올줄 알았지만, 깨진 부분들이 생각보다 많았음.
2. 코드를 추출해도, UI상으로 오류가 많이 발생하였음.
3. 말 그대로, UI 목업으로는 괜찮지만 실질적으로 필요한 부분들은 부족한 사항들이 많았음.
   
=> 별로 좋은 프로그램이 아닌 것 같아서 사용 포기
